# Steps to Access the Website

1. Go to GitHub and download the source code.
2. Set up the project in XAMPP. Make sure that MySQL is running on **port 3306**.
3. Open the landing page by visiting:

   ```
   http://localhost/sms/sms/index.php
   ```
4. Click the **"Get Started"** button.
5. You will be redirected to:

   ```
   http://localhost/sms/auth/Login.php
   ```
6. Choose the portal or website role you want to access.
7. Log in using one of the accounts below and explore the system.
