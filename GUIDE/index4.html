Lgu2Minutes

🚀

Getting Started

A comprehensive system developed for barangay logistics and community monitoring with an emphasis on accessibility.

🔒

The live system is currently under maintenance. Please check the GitHub repository for updates.

🧭

Sidebar Navigation

Navigation Modules

The system provides access to the following modules through the left sidebar:

📊

Dashboard

Main overview showing real-time meeting and agenda management, meeting status, and attendance trends.

🕒

Attendance

Manage and track daily attendance records and performance.

👥

Committees

Directory and hub for handling committee assignments and members.

📝

Minutes

Create, edit, and access official minutes of the meetings and sessions.

📂

Post-Session

Handle records, monitoring, and follow-ups after meeting sessions.

🗳️

Voting

Manage resolutions and track real-time voting during sessions.

💰

Incentives

Track payroll, salaries, and other member incentives and compensation.
